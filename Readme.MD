# 💪 PowerForge Gym

A modern, responsive gym website built with pure HTML, CSS, and JavaScript — no frameworks needed.

---

## 🌐 Live Preview

> Open `index.html` in your browser to view the project locally.

---

## 📸 Features

- ✅ Fully responsive design (mobile & desktop)
- ✅ Smooth mobile navigation with hamburger menu
- ✅ Hero section with call-to-action
- ✅ "Why Choose Us" advantages section
- ✅ Membership pricing cards (6 plans)
- ✅ About Us section with image slider
- ✅ Certified Trainers showcase (8 trainers)
- ✅ Footer with contact info & social links
